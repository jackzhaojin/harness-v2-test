# Simple Todo List App

## Problem

Build a simple, functional todo list web application that runs in a browser. Pure HTML/CSS/JS, no frameworks, no build tools.

## Requirements

1. **index.html** with:
   - An input field to add new todos
   - An "Add" button
   - A list displaying all todos
   - Each todo should have a checkbox to mark as complete
   - Each todo should have a delete button

2. **style.css** with:
   - Clean, modern styling
   - Proper spacing and layout
   - Visual distinction for completed items (strikethrough + muted color)

3. **app.js** with:
   - Ability to add new todos
   - Ability to toggle completion status
   - Ability to delete todos
   - Store todos in memory (no persistence required)

## Definition of Done

- [ ] All three files (index.html, style.css, app.js) exist and are linked correctly
- [ ] Can add a new todo item
- [ ] Can mark a todo as complete (visual indication)
- [ ] Can delete a todo
- [ ] App loads without console errors
- [ ] Styling looks presentable (not default browser styles)
