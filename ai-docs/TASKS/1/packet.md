# Task 1: Project scaffolding with Vite, React, TypeScript, and Tailwind

## Goal
Initialize the complete project foundation including Vite setup, React 18 with TypeScript strict mode, Tailwind CSS configuration, and all required dependencies. Establish the base file structure and configuration files as defined in the architecture.

## Acceptance Criteria
- [ ] Vite project initialized with React TypeScript template
- [ ] TypeScript strict mode enabled in tsconfig.json with no implicit any
- [ ] Tailwind CSS installed and configured with dark mode class strategy
- [ ] postcss.config.js configured for Tailwind
- [ ] index.css imports Tailwind base, components, and utilities
- [ ] React Router v6 installed and basic BrowserRouter configured in App.tsx
- [ ] Lucide React icons package installed
- [ ] Recharts library installed for data visualization
- [ ] Playwright installed for E2E testing with basic config
- [ ] npm run dev starts development server without errors
- [ ] npm run build succeeds without errors
- [ ] Base folder structure created: src/types, src/data, src/context, src/hooks, src/components, src/pages
- [ ] Empty placeholder files created for main architectural pieces
