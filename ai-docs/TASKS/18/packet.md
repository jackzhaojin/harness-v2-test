# Task 18: SlideOver panel component

## Goal
Build the reusable SlideOver panel component used for task details and potentially other uses. Includes slide animation, focus trap, and accessibility features.

## Acceptance Criteria
- [ ] SlideOver component slides in from right side of viewport
- [ ] Semi-transparent backdrop behind panel
- [ ] Panel has header area for title and close button
- [ ] Panel has scrollable body area for content
- [ ] Panel has optional footer area for actions
- [ ] Close button (X) in header closes panel
- [ ] Clicking backdrop closes panel
- [ ] Pressing Escape key closes panel
- [ ] Focus trapped inside panel when open
- [ ] Smooth slide animation on open/close (300ms)
- [ ] Panel renders via React Portal
- [ ] Component accepts onClose callback prop
- [ ] Works correctly in both light and dark modes
- [ ] TypeScript props interface defined

## Required Checks
- [ ] Smoke: App loads without errors
